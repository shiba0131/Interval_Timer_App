# インターバルタイマー 仕様書

## 概要
ユーザーが設定した2つの時間（例：作業時間と休憩時間）を交互に繰り返すインターバルタイマーのWebアプリケーション。
設定した時間が終了する3秒前から、カウントダウン音（ピッ、ピッ、ピッ、ピーー）を鳴らしてユーザーに知らせる機能を備える。

## 主な機能
1. **時間設定機能**
   - インターバル1の時間設定（分・秒）
   - インターバル2の時間設定（分・秒）
   - デフォルトは「10分」と「1分」に設定
2. **タイマー制御機能**
   - スタート、一時停止、リセット機能
3. **ループ機能**
   - インターバル1とインターバル2を自動的に交互に連続して繰り返す
   - 現在どちらのインターバルを実行中か視覚的に色やテキストで表示
4. **サウンド通知機能**
   - 残り3秒、2秒、1秒のタイミングで短い「ピッ」という音（予鈴）
   - 0秒（インターバル切り替わり）のタイミングで長い「ピーー」という音（本鈴）

## 技術スタック
- **構造**: HTML5
- **スタイル**: Vanilla CSS (ダークモード対応、モダンでスタイリッシュなUI)
- **ロジック**: Vanilla JavaScript (requestAnimationFrameによる正確な時間管理)
- **サウンド**: Web Audio API（ブラウザ標準機能により外部音声ファイルなしで動的に音を生成）
