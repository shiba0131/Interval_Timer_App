基本の4コマンド (新規作成・更新時)
ステージング（追加）: 
①git add .
変更したファイルをすべてステージングエリアに追加します。
コミット（記録）: 
②git commit -m "メッセージ"
※
変更内容をローカルリポジトリに保存します。
ブランチの指定 (初回のみ必要): git branch -M main
メインブランチの名前を「main」にします。
プッシュ（送信）: 
③git push origin main
git push -u origin main（-u 初回）
ローカルの変更をGitHub（リモート）へ送信します。
